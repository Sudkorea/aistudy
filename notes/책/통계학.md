## **먼저: εᵢ가 뭔가?**

### **정의**

```
진짜 모형: Y = β₀ + β₁X₁ + β₂X₂ + ε
                                ↑
                              오차항

ε = Y - E[Y|X]
  = 관측값 - 참 기댓값
```

**의미**:

- 모형으로 설명 **못하는** 부분
- 측정 오차, 누락 변수, 무작위성 등
- **우리가 관측 못함** (모집단 개념)

---

### **εᵢ vs eᵢ (잔차)**

|구분|ε (오차)|e (잔차)|
|---|---|---|
|정의|Y - E[Y\|X]|Y - Ŷ|
|관측|✗ 불가능|✓ 가능|
|위치|모집단|표본|
|예시|진짜 오차|추정 오차|

```
ε: 신만 아는 진짜 오차
e: 우리가 계산한 잔차
```

## **1. 이분산성 (Heteroscedasticity)**

### **정의**

```
Var(εᵢ) = σ²ᵢ (각각 다름!)

i=1: Var(ε₁) = 10
i=2: Var(ε₂) = 25
i=3: Var(ε₃) = 40
...
```

**등분산 (좋음)**:

```
Var(εᵢ) = σ² (모두 같음)
```

---

### **시각화**

**등분산** (정상):

```
잔차│  •  •  •
    │ • • • • •
    │• • • • • •
    └─────────→ Ŷ
    (균일하게 분포)
```

**이분산** (문제):

```
잔차│         • •
    │       • • •
    │     • • • •
    │   • • • • •
    │ • • • • • • •
    └─────────→ Ŷ
    (깔때기 모양!)
```

---

### **언제 발생?**

**횡단면 데이터**에서 흔함:

```
예1: 소득과 소비
- 저소득층: 소비 변동 작음 (다들 검소)
- 고소득층: 소비 변동 큼 (사치품 다양)

예2: 기업 규모와 매출
- 소기업: 매출 변동 작음
- 대기업: 매출 변동 큼
```

---

### **결과**

```
✓ 불편성 유지
✗ 효율성 상실
✗ 표준오차 추정 틀림
✗ t-검정 부정확
```

---

### **진단**

- **Breusch-Pagan 검정**
- **White 검정**
- 잔차 플롯 (깔때기 모양?)

---

### **해결**

- WLS (가중최소제곱)
- 로버스트 표준오차
- 로그 변환

## **2. 자기상관 (Autocorrelation)**

### **정의**

```
Cov(εᵢ, εⱼ) ≠ 0 (i≠j)

오차끼리 상관관계!

εᵢ₋₁이 크면 → εᵢ도 크다
```

**독립성 (좋음)**:

```
Cov(εᵢ, εⱼ) = 0 (i≠j)
```

---

### **이 문제의 예시 (1차 자기상관)**

```
εᵢ = ρεᵢ₋₁ + vᵢ

ρ: 자기상관계수
vᵢ: 순수 무작위 오차

εᵢ가 이전 오차 εᵢ₋₁에 의존!
```

**양의 자기상관 (ρ > 0)**:

```
어제 오차 ↑ → 오늘 오차 ↑ (지속)
```

**음의 자기상관 (ρ < 0)**:

```
어제 오차 ↑ → 오늘 오차 ↓ (진동)
```

---

### **시각화**

**독립** (정상):

```
ε│ +  -  +  -  +
  │  -  +  -  +  -
  └───────────→ t
  (무작위)
```

**양의 자기상관** (문제):

```
ε│  + + +
  │        - - -
  │  + + +
  └───────────→ t
  (패턴 있음!)
```

---

### **언제 발생?**

**시계열 데이터**에서 흔함:

```
예1: 주가
- 어제 상승 → 오늘도 상승 (추세)

예2: GDP 성장률
- 작년 높음 → 올해도 높음 (경기 지속)

예3: 날씨
- 어제 더움 → 오늘도 더움
```

---

### **결과**

```
✓ 불편성 유지
✗ 효율성 상실
✗ 표준오차 과소추정
✗ t값 과대평가 (거짓 유의성!)
```

---

### **진단**

- **Durbin-Watson (DW) 검정**
    - DW ≈ 2: 독립
    - DW < 2: 양의 자기상관
    - DW > 2: 음의 자기상관
- **Breusch-Godfrey 검정**

---

### **해결**

- 시차변수 추가
- Cochrane-Orcutt
- Newey-West 표준오차

## **εᵢ 최종 정리**

```

특징:

5. εᵢ ~ N(0, σ²) (정규성)
```

## **1. k의 의미**

### **k = 경계값 (임계값)**

```
기각역: ΣXi² / σ² > k

k의 역할:
"이 값을 넘으면 기각"하는 기준선

α에 맞춰 k 결정:
P(ΣXi²/σ² > k | H₀) = α
```

---

### **k 찾는 법**

```
1. H₀ 하에서 통계량 분포 확인
   ΣXi²/σ² ~ χ²(n)

2. α 설정 (예: 0.05)

3. 카이제곱표에서 찾기
   P(χ²(n) > k) = 0.05
   → k = χ²(n, 0.05)

예: n=10, α=0.05
   k = 18.307
```

---

### **당신의 질문 답변**

> "n=4,5 / n=6,7일 때 단순 X 제곱해서 대소 비교?"

**아니요!**

```
최량기각역 형태:
ΣXi² > k × σ²

k는 카이제곱 분포에서 찾음!

단순 비교 ✗
→ 분포 기반 계산 ✓
```

## **2. 검정 분포 완전 정리**

### **카이제곱 (χ²) 분포**

**정의**:

```
Z₁, Z₂, ..., Zₙ ~ N(0,1) (독립)

χ² = Z₁² + Z₂² + ... + Zₙ²
   ~ χ²(n)
```

**검정에서 사용**:

#### **① 분산 검정**

```
H₀: σ² = σ₀²

검정통계량:
χ² = (n-1)S² / σ₀²
   ~ χ²(n-1)

"표본분산이 귀무가설 분산과 맞나?"
```

#### **② 적합도 검정**

```
관측도수 vs 기대도수

χ² = Σ(O - E)² / E
   ~ χ²(k-1)
```

#### **③ 독립성 검정**

```
분할표에서 독립성
χ² ~ χ²((r-1)(c-1))
```

---

### **t 분포**

**정의**:

```
Z ~ N(0,1)
V ~ χ²(n)
Z, V 독립

t = Z / √(V/n)
  ~ t(n)
```

**검정에서 사용**:

#### **① 모평균 검정 (σ² 모를 때)**

```
H₀: μ = μ₀

검정통계량:
t = (X̄ - μ₀) / (S/√n)
  ~ t(n-1)

언제: σ² 모를 때!
     (σ² 알면 Z-검정)
```

#### **② 두 평균 차이 (등분산)**

```
H₀: μ₁ = μ₂

t = (X̄ - Ȳ) / √(Sp²(1/n₁ + 1/n₂))
  ~ t(n₁+n₂-2)
```

---

### **F 분포**

**정의**:

```
U ~ χ²(n₁)
V ~ χ²(n₂)
U, V 독립

F = (U/n₁) / (V/n₂)
  ~ F(n₁, n₂)
```

**검정에서 사용**:

#### **① 두 분산 비교**

```
H₀: σ₁² = σ₂²

F = S₁² / S₂²
  ~ F(n₁-1, n₂-1)

"두 집단 분산이 같나?"
```

#### **② 회귀분석 (전체 모델)**

```
H₀: β₁ = β₂ = ... = 0

F = MSR / MSE
  ~ F(k, n-k-1)

"모델이 의미있나?"
```

#### **③ 분산분석 (ANOVA)**

```
H₀: μ₁ = μ₂ = ... = μₖ

F = MSB / MSW
  ~ F(k-1, n-k)

"여러 그룹 평균이 같나?"
```

## **비교표 (완전판)**

|분포|정의|모수 검정|사용 조건|
|---|---|---|---|
|**Z (정규)**|표준정규|평균 (σ² 알 때)|σ² 알고, n 큼|
|**t**|Z/√(χ²/n)|평균 (σ² 모를 때)|**σ² 모름**|
|**χ²**|ΣZ²|**분산**, 적합도|분산 검정|
|**F**|(χ²/n₁)/(χ²/n₂)|**분산 비**, 회귀|두 분산, ANOVA|

## **핵심 오해 수정**

### **당신의 이해**

> "t분포는 n < 30인 경우 정규분포 근사?"

**수정**:

```
✗ "n < 30이라서" t 쓰는게 아님
✓ "σ² 모를 때" t 쓰는 것!

n 작으면:
- σ² 알아도 모를 가능성 큼
- 그래서 t 많이 씀

하지만 핵심은 σ² 모름!
```

---

> "F분포는 공분산?"

**수정**:

```
✗ 공분산 아님
✓ 분산 비율!

F = σ₁² / σ₂²

공분산 = Cov(X,Y)
       ≠ F분포
```
