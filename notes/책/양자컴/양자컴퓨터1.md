# 2차 양자 혁명: 양자 정보 과학과 기술 강의

안녕하세요! 오늘은 **2차 양자 혁명(2nd Quantum Revolution)** 과 **양자 정보 과학 기술(Quantum Information Science & Technology)** 에 대해 함께 공부하겠습니다.

## 1강: 정보의 기본 단위 - 비트(Bit)

### 비트란 무엇인가?

**비트(bit)**는 "binary digit(이진 숫자)"의 줄임말로, 디지털 정보의 기본 단위입니다.

- **0 또는 1**의 두 가지 상태만 가능
- 예시: 흑/백, 위/아래, 좌/우, 남/여 등 **두 가지 반대되는 것**

### 비트의 활용

- **스무고개 게임**: 예/아니오(1/0) 질문 20개 = 20비트 → 2²⁰ = 1,048,576가지를 구별 가능
- **4지선다 문제** = 2비트 문제 (2² = 4)

## 2강: 양자역학의 탄생

### 양자(Quantum, 量子)의 의미

- **1900년 Max Planck**: 흑체복사 연구
    - 빛 에너지가 **연속적이 아닌 불연속적**이라는 발견
    - 이것이 "양자(quantum)"의 시작

### 주요 발전 과정

- **1905년 Einstein**: 광전효과 (노벨상 수상 업적)
- **1913년 Bohr**: 수소 원자의 전자 궤도 - 불연속적 궤도
- **1924년 de Broglie**: 물질파
- **1925년 Heisenberg**: 행렬역학
- **1926년 Schrödinger**: 파동역학
- **1927년 Heisenberg**: **불확정성 원리**

> 💡 **2025년은 IYQ(International Year of Quantum Science and Technology)**입니다!

## 5강: 큐비트(Qubit) - 양자 정보의 단위

### 비트 vs 큐비트

**고전 비트:**

- 0 **또는** 1
- 한 번에 하나의 상태

**큐비트(Quantum Bit):**

- 0 **과** 1이 **동시에** 존재
- **중첩 상태**

```
|ψ⟩ = α|0⟩ + β|1⟩
```

여기서:

- |α|² + |β|² = 1
- |α|²: 측정 시 0이 나올 확률
- |β|²: 측정 시 1이 나올 확률

### 블로흐 구(Bloch Sphere)

큐비트의 순수 상태는 3차원 구 표면의 한 점으로 표현됩니다.

```
|ψ⟩ = e^(iγ)(cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩)
```

## 6강: 양자 게이트와 양자 회로

### 단일 큐비트 게이트

**하다마드(Hadamard) 게이트:**

```
H = 1/√2 [1   1]
         [1  -1]

H|0⟩ = |+⟩ = (|0⟩ + |1⟩)/√2
H|1⟩ = |−⟩ = (|0⟩ − |1⟩)/√2
```

**파울리 게이트:**

- X (비트 반전): |0⟩ ↔ |1⟩
- Y, Z 게이트

### 2큐비트 게이트: CNOT (제어-NOT)

```
CNOT|a⟩|b⟩ = |a⟩|a⊕b⟩
```

- 첫 번째 큐비트(제어)가 1이면 두 번째 큐비트(타겟) 반전

## 7강: 양자 컴퓨터의 위력

### 계산 복잡도 비교

**소인수분해 문제:**

- 고전 컴퓨터: 2^(N^(1/3))
- **양자 컴퓨터: N³** (쇼어 알고리즘)

**데이터 검색:**

- 고전: N/2
- **양자: √N** (그로버 알고리즘)

### 양자 병렬성

N개의 큐비트로 2^N개의 상태를 **동시에** 표현:

```
4 큐비트 = 2⁴ = 16가지 상태를 동시에!
```

## 1단계: SWAP을 CNOT으로 분해하기

### SWAP = 3개의 CNOT!

```
SWAP = CNOT₁₂ · CNOT₂₁ · CNOT₁₂
```

**회로도:**

```
  1 ───●───⊕───●───
        │   │   │
  2 ───⊕───●───⊕───
```

### 실제로 확인해봅시다:

**|01⟩에 적용:**

```
|01⟩ 
  → CNOT₁₂ → |01⟩  (1이 0이라 변화 없음)
  → CNOT₂₁ → |11⟩  (2가 1이라 1 반전)
  → CNOT₁₂ → |10⟩  (1이 1이라 2 반전)
✓ 교환 완료!
```

**|10⟩에 적용:**

```
|10⟩
  → CNOT₁₂ → |11⟩  (1이 1이라 2 반전)
  → CNOT₂₁ → |11⟩  (2가 1이라 1 반전... 어? 1→0)
  실제로는 |01⟩
  → CNOT₁₂ → |01⟩
✓ 교환 완료!
```

따라서 **SWAP은 기본 게이트가 아닙니다!**

## 2단계: 보편 양자 게이트 집합 (Universal Gate Set)

### 양자 컴퓨팅의 기본 정리:

**임의의 양자 연산은 다음으로 근사 가능:**

```
1. 모든 단일 큐비트 유니터리 (Single-qubit gates)
2. 하나의 2큐비트 얽힘 게이트 (Entangling gate)
```

### 대표적인 보편 게이트 집합:

**집합 1: {H, T, CNOT}**

- H (Hadamard)
- T (π/8 게이트)
- CNOT

**집합 2: {Rz(θ), Ry(θ), CNOT}**

- 임의의 회전

**집합 3: {U(θ,φ,λ), CNOT}**

- 일반 단일 큐비트 게이트

## 3단계: 대수적 접근 - 리 군(Lie Group) 이론

### 단일 큐비트: SU(2) 군

**유니터리 행렬 조건:**

```
U†U = I  (유니터리)
det(U) = 1  (특수 유니터리)
```

**SU(2)의 생성자 (Generators):**

```
σx = [0 1]    σy = [0 -i]    σz = [1  0]
     [1 0]         [i  0]         [0 -1]
```

파울리 행렬!

**일반적 SU(2) 원소:**

```
U(θ,φ,λ) = e^(-iφ)Rz(φ)Ry(θ)Rz(λ)

         = [cos(θ/2)           -e^(iλ)sin(θ/2)    ]
           [e^(iφ)sin(θ/2)     e^(i(φ+λ))cos(θ/2) ]
```

**차원:** SU(2)는 3차원 (θ, φ, λ)

---

### 2큐비트: SU(4) 군

**차원:** SU(4)는 **15차원**!

```
4×4 복소 행렬 → 16개 자유도
det=1 조건 → -1
유니터리 조건 → -8 (허미트성)
= 15차원
```

**문제:**

- 단일 큐비트 게이트만으로는: 2×3 = 6차원만 커버
- **9차원이 부족!** → 얽힘 게이트 필요

## 4단계: 얽힘 능력 (Entangling Power)

### 얽힘을 만들 수 있는가?

**정의:** 게이트 U가 **얽힘 게이트**라면:

```
∃ 분리 가능한 입력 |φ⟩⊗|χ⟩
such that U(|φ⟩⊗|χ⟩) = 얽힌 상태
```

### 예시:

**CNOT:**

```
CNOT (|+⟩⊗|0⟩) = CNOT · 1/√2(|0⟩+|1⟩)⊗|0⟩
                = 1/√2(|00⟩+|11⟩) = |Φ⁺⟩
```

✓ 얽힌 상태 생성! → **얽힘 게이트**

**SWAP:**

```
SWAP (|0⟩⊗|1⟩) = |1⟩⊗|0⟩
```

분리 가능 → 분리 가능

하지만 SWAP도 중첩 상태에는:

```
SWAP · 1/√2(|01⟩+|10⟩) = 1/√2(|10⟩+|01⟩)
```

이미 얽혀있던 것은 유지하지만 **새로 만들지는 못함**

## 5단계: Cartan 분해 이론

### 2큐비트 게이트의 수학적 분류

**Cartan-KAK 분해:**

```
U = (U₁⊗U₂) · exp[i(αXX + βYY + γZZ)] · (U₃⊗U₄)
```

여기서:

- U₁, U₂, U₃, U₄: 단일 큐비트 게이트 (local)
- 가운데 항: **non-local 얽힘** (α, β, γ로 특징지어짐)

### 얽힘 계수 (α, β, γ):

**CNOT:**

```
CNOT = exp[iπ/4(XX + YY - ZZ)]
```

(적절한 단일 큐비트 게이트 곱하면)

→ α = β = π/4, γ = -π/4 → **non-trivial entanglement!**

**SWAP:**

```
SWAP = exp[iπ/2(XX + YY + ZZ)]
```

→ 모든 계수가 같음 → 특수한 경우 → CNOT 3개로 분해 가능

## 6단계: 게이트의 "크기" 정의

### 여러 관점:

**1. 회로 복잡도 (Circuit Complexity):**

```
SWAP = 3 CNOT
```

→ CNOT이 더 기본적

**2. 얽힘 능력 (Entangling Power):**

```
E(CNOT) = 2 ebits
E(SWAP) = 0 ebits
```

→ CNOT이 더 "강력"

**3. 대수적 생성 (Algebraic Generation):**

- {단일큐비트 + CNOT} → SU(4) 전체 생성 가능
- {단일큐비트 + SWAP} → SU(4) 일부만 생성

→ CNOT이 더 기본적

## 7단계: 보편성의 수학적 정리

### Solovay-Kitaev 정리:

**내용:** 보편 게이트 집합 G = {g₁, g₂, ...}가 있으면, 임의의 n-큐비트 유니터리 U를 ε 오차로 근사하는데 필요한 게이트 수:

```
O(log^c(1/ε))  where c ≈ 2
```

### 2큐비트 보편성 정리:

**정리 (Barenco et al., 1995):**

```
게이트 U가 보편적 ⟺ U는 얽힘 게이트 ∧ U ∉ Local Clifford
```

**CNOT:**

- 얽힘 게이트 ✓
- Local Clifford가 아님 ✓
- → **보편적!**

**SWAP:**

- 얽힘 게이트 ✗
- → 보편적 아님

## 8단계: 교환자와 리 대수

### 리 대수 관점:

**교환자 (Commutator):**

```
[A, B] = AB - BA
```

**중요한 성질:**

```
[Hlocal, CNOT] ≠ 0
[Hlocal, SWAP] = ?
```

### 계산해보기:

**Z₁과 CNOT:**

```
[Z⊗I, CNOT] = (Z⊗I)CNOT - CNOT(Z⊗I) ≠ 0
```

→ CNOT은 local과 **다른 부분공간** 생성

**Z₁과 SWAP:**

```
[Z⊗I, SWAP] = (Z⊗I)SWAP - SWAP(Z⊗I)
```

계산하면... 복잡하지만 결과적으로: SWAP은 단일 큐비트 게이트의 **대칭화된 조합**

## 9단계: 대수적 계층 구조

### 양자 게이트의 대수적 위계:

```
레벨 0: 단일 큐비트 게이트 (SU(2)⊗SU(2)) 
        ↓ 차원: 6

레벨 1: Controlled 게이트 (CNOT, CZ, ...)
        ↓ 추가 차원: +9
        → 보편성 달성! (전체 SU(4))

레벨 2: 특수 조합 (SWAP, Toffoli, ...)
        → 레벨 0+1로 구성 가능
```

### 기하학적 해석:

SU(4)를 manifold로 보면:

```
- 단일 큐비트: 6차원 부분공간
- +CNOT: 전체 15차원 접근 가능
- SWAP: 특정 궤도(orbit) 위를 움직임
```

## 10단계: 실용적 관점

### 실제 양자 컴퓨터 구현:

**하드웨어별 native gate:**

**초전도 큐비트 (Google, IBM):**

```
Native: √iSWAP, CZ
CNOT = √iSWAP + 단일큐비트
```


```
Native: CZ (Rydberg blockade)
```

→ 물리적으로는 다른 게이트가 "기본"!
