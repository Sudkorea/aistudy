## Abstract
1. Object detection 성능이 정체되어 있었던 주요 원인
    - HOG나 SIFT같은 낮은 수준의 특징들에 의존
    - 컴퓨터 비전에서 획기적인 발전을 이끈 CNN을 detection에 적용하지 못함
2. 주요 제안 내용
    - R-CNN(Regions with CNN features)이라는 새로운 방법 제시
    - PASCAL VOC 2012에서 mAP 53.3% 달성 (기존 최고 성능 대비 30% 향상)
    - ImageNet ILSVRC2013 detection에서도 우수한 성능 입증

## Introduction

1. 연구 배경
    - CNN이 이미지 분류에서는 큰 성공을 거둠
    - 하지만 object detection은 여전히 전통적인 방식(sliding window)에 의존
    - sliding window approach는 계산 비용이 매우 높음
2. R-CNN의 핵심 아이디어
    - "recognition using regions" 패러다임 채택
    - selective search로 2000개 정도의 category-independent region proposals 생성
    - 각 region을 CNN으로 처리하여 fixed-length feature vector 추출
    - linear SVM으로 각 region 분류
3. 주요 기술적 기여
    - large CNN을 detection에 성공적으로 적용
    - labeled data가 부족할 때 supervised pre-training과 domain-specific fine-tuning이 효과적임을 입증
    - CNN feature를 이용한 bounding box regression 방법 제안
4. 성능
    - VOC 2012 테스트셋에서 mAP 53.3% 달성
    - 200개 클래스가 있는 ILSVRC2013에서도 우수한 성능 입증
    - computation sharing을 통해 실행 시간도 개선

## Conclusion

1. 주요 성과
- PASCAL VOC 2012에서 mAP 53.3% 달성 (기존 성능보다 크게 향상)
- 200개 클래스를 가진 ILSVRC2013 detection dataset에서도 높은 성능 입증
- ILSVRC2013의 웹 이미지와 같은 도전적인 환경에서도 효과적임을 보임

2. 핵심 기술적 발견
- Object detection에서도 CNN이 매우 효과적임을 입증
    - 특히 large CNN의 성능이 두드러짐
    - detection을 위한 domain-specific한 fine-tuning이 중요
- CNN feature가 기존의 전통적 특징들(HOG 등)보다 훨씬 우수함을 보임
- 적은 양의 annotated detection data로도 좋은 성능 달성 가능
    - supervised pre-training + fine-tuning 전략이 효과적

3. 한계점 및 개선 가능성
- Selective search가 생성하는 region proposals의 품질에 성능이 의존적
- Detection pipeline이 다소 복잡하고 여러 단계로 구성됨
- Test time 성능이 아직 최적화되지 않음
- Feature caching 등으로 속도를 개선할 여지있음
